Project 2

This is a simple task management web application that allows users to add tasks with a title and a summary.

Users can enter a task title and a task summary in the input fields.

A button labeled "Add" allows them to add the task.

When the user clicks the "Add" button, the addTask() function is executed.

It retrieves input values, checks if they are empty, and then creates a new task card.

The new task card is added to the task list dynamically.
